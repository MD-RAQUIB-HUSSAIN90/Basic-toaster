# 🍞 JavaScript Toast Notification


<h3>LIVE DEMO :- </h3>

A simple and customizable Toast Notification component built using **Vanilla JavaScript**, **HTML**, **CSS**, and **Tailwind CSS**.

This project demonstrates how to create reusable toast notifications using **Higher Order Functions**, **Closures**, and **DOM Manipulation**.

---

## ✨ Features

- 📍 Custom Position (Top/Bottom & Left/Right)
- 🎨 Light & Dark Theme
- ⏱️ Custom Display Duration
- 🔁 Reusable Toast Function
- 📦 Lightweight and Easy to Use
- ⚡ Built with Vanilla JavaScript
- 🎯 Clean and Minimal UI

## 🚀 How It Works

Create a toaster instance by passing a configuration object.

```javascript
let toaster = createToaster({
  positionX: "right",
  positionY: "bottom",
  theme: "light",
  duration: 3,
});

toaster("Download Done");
```

---

## ⚙️ Configuration Options

| Property | Values |
|----------|---------|
| positionX | left / right |
| positionY | top / bottom |
| theme | light / dark |
| duration | Time in seconds |
